# **App Name**: StockSync

## Core Features:

- Login Authentication: Secure user login with email and password, including 'Remember Me' functionality and a 'Forgot Password?' link.
- Dashboard Overview: Display key inventory metrics such as total products, categories, stock alerts, and monthly revenue.
- Inventory Movement Chart: Visualize stock trends over time with a line chart, showing weekly stock movement.
- Stock Level Prediction: AI tool to suggest target inventory reorder values based on sales and stock levels with progress bar indicator.
- Product Management: Table to display a sortable, paginated list of products with details like name, category, stock, price, and status, including product images.
- Sidebar Navigation: Provide an intuitive sidebar menu for easy navigation between Dashboard, Products, Categories, Suppliers, Purchase Orders, Sales/Issue Goods, Reports, Settings, and Logout.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) for a professional and trustworthy feel.
- Background color: Light Grey (#F7FAFC) for a clean, modern aesthetic.
- Accent color: Sky Blue (#60A5FA) for interactive elements and highlights.
- Body text: 'PT Sans', a humanist sans-serif, will be used for the main content for a balance of modern and warm feel.
- Headline text: 'Playfair', a modern serif, will be used for the headlines or titles to bring elegant, high-end feel; using 'PT Sans' as the body
- Use modern, minimalistic icons from a consistent set to represent menu items and actions.
- Employ rounded cards, soft shadows, and consistent spacing inspired by the reference image for a clean, professional layout.
- Use subtle transitions and animations to enhance user experience and provide feedback on interactions.
